Extension Guide

Important

The item display settings you have set on the ekizo.mandarake.co.jp/shop/ website will affect this extension. The following options must be set in order for this extension to function properly (default settings):
  • Order of Display must be set to New Arrival
  • Sort of Display must be set to Descending Order

Recommended

These options can be set to improve performance.
  • Number of Items set to 50 or 100. This will reduce the number of pages the extension needs to search through.
  • Sold Out set to Show All Including Sold Out Items. The extension keeps track of the last newest item listing it found in order to know when to stop searching. If Sold Out is set to Show Only Available Items and the last newest item is sold out, the extension will miss it and will be forced to search through all the pages again (up to the search page limit). The extension will only notify you of in-stock items regardless of this setting.

Keywords

  • Keywords/phrases must be between 3 - 60 characters.
  • You can have up to 30 keywords/phrases.
  • They are case-insensitive. e.g. ALTER is the same as alter.
  • Phrases will match with an item description as long as all of its words appear somewhere in the description. e.g. Key phrase alter saber nakano will match item description [Nakano] Alter - Saber Lily, but not [Nakano] Good Smile - Saber.
  • Enclose a phrase with double or single quotes for precise matching. e.g. nakano "saber alter" will match [Nakano]Good Smile - Saber Alter, but it will not match [Nakano] Saber Maid Alter.
  • Enclose a word or phrase with vertical bars (|) if you want to exclude it. e.g. saber |figma| will match Good Smile - Saber but not Max Factory - Saber Figma.
