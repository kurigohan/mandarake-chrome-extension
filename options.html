Mandrake Tracker Options


Please mote when adding keywords:

  • Keywords can be up to 50 characters long.
  • They are case-insensitive. e.g. 'ALTER' is the same as 'alter'.
  • Word order matters in phrases. e.g. 'Good Company Smile' is not the same as 'Good Smile Company'.
  • Phrases will match with an item description as long as all the words in the phrase appear somewhere in the description (order still matters). e.g. The key phrase 'nakano saber figure' will match with '[Nakano] Alter Saber Lily 1/8 Figure'.
