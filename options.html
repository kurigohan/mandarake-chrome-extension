Keyword guide

  • Keywords/phrases can be up to 50 characters long.
  • You can store up to 20 keywords/phrases.
  • They are case-insensitive. e.g. 'ALTER' is the same as 'alter'.
  • Phrases will match with an item description as long as all of its words appear in the description (even if its part of another word) and are in the same order as they are in the phrase. e.g. Key phrase 'nakano alter saber' will match item description '[Nakano] Alter - Saber Lily', but not '[Nakano] Good Smile - Saber Alter'.
  • Enclose a phrase with double quotes for exact matching.e.g. 'Nakano "Saber Alter" ' will match '[Nakano]Good Smile - Saber Alter', but it will not match '[Nakano] Saber Maid Alter'.
