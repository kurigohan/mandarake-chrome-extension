MANDRAKE TRACKER OPTIONS




Please note when adding keywords:

  • Keywords or phrases can be up to 50 characters long.
  • You can store up to 20 keywords or phrases.
  • They are case-insensitive. e.g. 'ALTER' is the same as 'alter'.
  • Phrases will match with an item description as long as all the words in the phrase appear somewhere in the description. e.g. The key phrase 'nakano saber figure' will match with '[Nakano] Alter Saber Lily 1/8 Figure'.
  • Word order matters in phrases. e.g. 'Good Company Smile' is not the same as 'Good Smile Company'.
